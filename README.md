
本项目用于研究多媒体相关应用，
期望实现：
1、流媒体播放。
2、播放进度和缓存进度监听显示（优化实现播放时间和剩余时间显示）。
3、视频下载，支持文件断点下载。
4、播放同步缓存。
5、本地视频资源管理。
6、本地最大缓存资源数、资源缓存最长生命期设定、一键清理所有缓存资源。（文件管理借鉴SDWebImage图片资源管理方案）

已经实现功能：
1、在线播放。
2、播放进度和缓存进度监听显示。
